## 🆘 БЫСТРОЕ РЕШЕНИЕ ПРОБЛЕМ:

| Проблема | Решение |
|----------|---------|
| Docker не открывается | `open -a Docker` или найти в Applications |
| MongoDB не подключается | `docker-compose down && docker-compose up -d` |
| Port 3000 занят | `lsof -i :3000 && kill -9 [PID]` |
| Port 5000 занят | `lsof -i :5000 && kill -9 [PID]` |
| Module errors | `cd frontend && npm cache clean --force && npm install` |
| Backend медленный | MongoDB может еще загружаться, подождите 30 сек |
