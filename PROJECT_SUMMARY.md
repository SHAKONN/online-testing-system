## 🗄️ БАЗА ДАННЫХ

### 4 Основные коллекции:

1. **Users** (Пользователи)
   - Хранит профили (name, email, password, role)
   - Статистику (totalTests, averageScore)

2. **Questions** (Вопросы)
   - Текст с 4 вариантами ответа
   - Категория (8 предметов)
   - Сложность (easy, medium, hard)
   - Объяснение правильного ответа

3. **Tests** (Тесты)
   - Название и описание
   - Категория и время
   - Масив ID вопросов
   - Статистика (попытки, средний результат)

4. **Results** (Результаты)
   - Сохраняет все ответы пользователя
   - Процент, баллы, время
   - Дата и время прохождения

**Всего индексов:** 15+ для оптимизации запросов

## 🌐 API ENDPOINTS

### Публичные (без авторизации):
```
POST   /api/auth/register          - Регистрация
POST   /api/auth/login             - Вход
GET    /api/tests                  - Список тестов
GET    /api/results/leaderboard/top - Лидерборд
```

### Защищенные (требуется JWT):
```
GET    /api/auth/profile           - Профиль
GET    /api/tests/category/:cat    - Тесты по категории
GET    /api/tests/:id/questions    - Тест с вопросами
POST   /api/results/submit         - Отправить результаты
GET    /api/results/my-results     - Мои результаты
GET    /api/results/:id            - Результат по ID
```

### Admin endpoints:
```
POST   /api/questions              - Создать вопрос
PUT    /api/questions/:id          - Редактировать вопрос
DELETE /api/questions/:id          - Удалить вопрос

POST   /api/tests                  - Создать тест
PUT    /api/tests/:id              - Редактировать тест
DELETE /api/tests/:id              - Удалить тест

GET    /api/results/stats/admin    - Статистика
```

## 🚀 БЫСТРЫЙ СТАРТ (5 шагов)

### 1. Установка
```bash
cd backend && npm install
cd ../frontend && npm install
```

### 2. Настройка MongoDB
```bash
# macOS: brew services start mongodb-community
# Linux: sudo service mongod start
# Windows: net start MongoDB
```

### 3. .env файлы
```bash
# backend/.env
MONGODB_URI=mongodb://localhost:27017/testing-system
JWT_SECRET=your-secret-key
PORT=5000

# frontend/.env
REACT_APP_API_URL=http://localhost:5000/api
```

### 4. Загрузка данных
```bash
cd backend
npm run seed
```

### 5. Запуск
```
# Терминал 1:
cd backend && npm run dev

# Терминал 2:
cd frontend && npm start
```

✅ **Готово!** Перейдите на http://localhost:3000

```
   ↓

   ↓
4. Отвечает на вопросы (таймер работает)
   ↓
5. Завершает тест (автоматически при истечении времени)

   ↓
7. Просматривает разбор всех ответов
   ↓
8. История сохраняется в БД
   ↓
9. Профиль обновляется (средняя оценка, количество тестов)
```

## ⚠️ ПРИМЕЧАНИЯ

1. При развертывании в production:
   - Измените `JWT_SECRET` на сложный ключ
   - Используйте переменные окружения
   - Включите HTTPS
   - Установите rate limiting

2. MongoDB:
   - Локально: `mongodb://localhost:27017`
   - Atlas: Получите connection string на mongodb.com

3. Переменные окружения не должны быть в коде!
