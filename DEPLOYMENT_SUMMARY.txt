🎯 БЫСТРЫЙ СТАРТ (ВЫБЕРИТЕ 1):

ВАРИАНТ A: RENDER.COM (Самый простой!)
  1. Откройте https://render.com
  2. Нажмите "New" → "Web Service"
  3. Подключите GitHub репо
  4. Создайте Backend, Frontend, MongoDB
  5. git push и автоматический deploy!

ВАРИАНТ B: DOCKER (Универсальный!)
  git clone ваш-репо && cd проект
  docker-compose -f docker-compose.prod.yml up -d
  ✅ Готово!

🔐 ОБЯЗАТЕЛЬНЫЕ ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ:

Backend:
  NODE_ENV=production
  PORT=5001
  JWT_SECRET=your-secret-min-32-chars
  MONGODB_URI=mongodb+srv://...
  FRONTEND_URL=https://your-domain.com

Frontend:
  REACT_APP_API_URL=https://api.your-domain.com/api
